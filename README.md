# Cassiopeia E-commerce
Uma aplicação fictícia, que apenas ilusta uma e-commerce real. Assim, com a finalidade de ser exposta como experiências do dev, em outros termos, parte do portifólio. Este aplicação conterá algumas das diveras funcionalidades de um e-commerce, pois aplicações deste porte classificam-se como complexas e seria inviável produzi

## Requisitos
Entre as diversas funcionalidades de um e-commerce, se descrevem como necessário:
- Banners;
- Carrosséis;
- Página do produto (incluindo suas variações);
- Visualização de catálogo, podendo o filtrar por categoria/termo de busca;
- Mini carrinho;
- Checkout, incluindo:
  - Seleção de endereços;
  - Métodos de entrega;
  - Métodos de pagamentos.
